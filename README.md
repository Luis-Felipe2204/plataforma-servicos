# Plataforma de Serviços Locais

Projeto desenvolvido para conectar usuários a profissionais próximos, permitindo encontrar serviços de forma rápida e confiável.

## 📌 Objetivo
Criar uma plataforma onde usuários possam encontrar profissionais como eletricistas, encanadores, técnicos e outros prestadores de serviço próximos.

## 🚀 Tecnologias utilizadas
- Python
- FastAPI
- Banco de dados
- GitHub

## 📂 Estrutura do projeto

backend → API e lógica do sistema  
frontend → interface do usuário  
docs → documentação do projeto
